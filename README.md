## 项目简介

基于多高度传感器采集的风速及相关气象数据，构建机器学习模型预测未来时刻的风速。

预测目标：100米高度风速（Speed_100m）

---

## 模型

| 模型 | 说明 |
|------|------|
| Linear Regression | 线性回归，作为基准模型 |
| LSTM | 长短期记忆网络，捕获时间依赖 |
| Transformer | 基于自注意力机制，建模全局依赖 |

---

## 预测任务

| 任务 | 输入 | 输出 |
|------|------|------|
| 单步预测 | 8小时历史（48步） | 预测下一时刻（10分钟） |
| 多步预测A | 8小时历史（48步） | 预测未来1小时（6步） |
| 多步预测B | 8小时历史（48步） | 预测未来16小时（96步） |
